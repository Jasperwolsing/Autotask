# Get-AtwsTagGroup

## SYNOPSIS
This function get one or more TagGroup through the Autotask Web Services API.

## SYNTAX

### Filter (Default)
```
Get-AtwsTagGroup -Filter <String[]> [-GetReferenceEntityById <String>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### By_parameters
```
Get-AtwsTagGroup [-GetReferenceEntityById <String>] [-DisplayColor <String[]>] [-id <Nullable`1[]>]
 [-IsActive <Nullable`1[]>] [-IsSystem <Nullable`1[]>] [-Label <String[]>] [-NotEquals <String[]>]
 [-IsNull <String[]>] [-IsNotNull <String[]>] [-GreaterThan <String[]>] [-GreaterThanOrEquals <String[]>]
 [-LessThan <String[]>] [-LessThanOrEquals <String[]>] [-Like <String[]>] [-NotLike <String[]>]
 [-BeginsWith <String[]>] [-EndsWith <String[]>] [-Contains <String[]>] [-IsThisDay <String[]>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### Get_all
```
Get-AtwsTagGroup [-All] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
This function creates a query based on any parameters you give and returns any resulting objects from the Autotask Web Services Api.
By default the function returns any objects with properties that are Equal (-eq) to the value of the parameter.
To give you more flexibility you can modify the operator by using -NotEquals \[ParameterName\[\]\], -LessThan \[ParameterName\[\]\] and so on.

Possible operators for all parameters are:
 -NotEquals
 -GreaterThan
 -GreaterThanOrEqual
 -LessThan
 -LessThanOrEquals 

Additional operators for \[string\] parameters are:
 -Like (supports * or % as wildcards)
 -NotLike
 -BeginsWith
 -EndsWith
 -Contains

Properties with picklists are:
DisplayColor

## EXAMPLES

### EXAMPLE 1
```
Get-AtwsTagGroup -Id 0
Returns the object with Id 0, if any.
```

### EXAMPLE 2
```
Get-AtwsTagGroup -TagGroupName SomeName
Returns the object with TagGroupName 'SomeName', if any.
```

### EXAMPLE 3
```
Get-AtwsTagGroup -TagGroupName 'Some Name'
Returns the object with TagGroupName 'Some Name', if any.
```

### EXAMPLE 4
```
Get-AtwsTagGroup -TagGroupName 'Some Name' -NotEquals TagGroupName
Returns any objects with a TagGroupName that is NOT equal to 'Some Name', if any.
```

### EXAMPLE 5
```
Get-AtwsTagGroup -TagGroupName SomeName* -Like TagGroupName
Returns any object with a TagGroupName that matches the simple pattern 'SomeName*'. Supported wildcards are * and %.
```

### EXAMPLE 6
```
Get-AtwsTagGroup -TagGroupName SomeName* -NotLike TagGroupName
Returns any object with a TagGroupName that DOES NOT match the simple pattern 'SomeName*'. Supported wildcards are * and %.
```

### EXAMPLE 7
```
Returns any TagGroups with property DisplayColor equal to the <PickList Label>. '-PickList' is any parameter on .
```

### EXAMPLE 8
```
-NotEquals DisplayColor 
Returns any TagGroups with property DisplayColor NOT equal to the <PickList Label>.
```

### EXAMPLE 9
```
, <PickList Label2>
Returns any TagGroups with property DisplayColor equal to EITHER <PickList Label1> OR <PickList Label2>.
```

### EXAMPLE 10
```
, <PickList Label2> -NotEquals DisplayColor
Returns any TagGroups with property DisplayColor NOT equal to NEITHER <PickList Label1> NOR <PickList Label2>.
```

### EXAMPLE 11
```
, <PickList Label2> -Like TagGroupName -NotEquals DisplayColor -GreaterThan Id
An example of a more complex query. This command returns any TagGroups with Id GREATER THAN 1234, a TagGroupName that matches the simple pattern SomeName* AND that has a DisplayColor that is NOT equal to NEITHER <PickList Label1> NOR <PickList Label2>.
```
